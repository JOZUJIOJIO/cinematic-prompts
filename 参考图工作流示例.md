## 🛠️ 使用工具建议

### Midjourney 工作流
```bash
# 镜头1
[上传参考图] + 完整图片提示词 + --cref [参考图URL] --cw 100 --ar 16:9 --style raw --v 6

# 镜头2-5
[上传镜头1生成图] + 完整图片提示词 + --cref [镜头1 URL] --cw 80 --ar 16:9 --style raw --v 6
```

### Stable Diffusion 工作流
```
1. 使用 img2img 模式
2. 上传参考图，降噪强度 0.4-0.6
3. 使用完整图片提示词
4. 使用 ControlNet (OpenPose/Canny) 保持姿势一致性
5. Seed 固定或使用 -1 随机
```

### 图转视频工作流 (Runway Gen-3)
```
1. 上传生成的静态图作为首帧
2. 输入对应的图转视频提示词
3. 设置运动强度: 低 (3-5)
4. 时长: 4-7秒
5. 保持相同的风格预设
```

## ✅ 一致性检查清单

- [ ] 所有镜头使用相同的人物DNA
- [ ] 所有镜头使用相同的场景DNA
- [ ] 所有镜头使用相同的风格DNA
- [ ] DNA权重保持在 1.2-1.5 范围
- [ ] 变化层权重保持在 0.8-1.0 范围
- [ ] 相邻镜头视觉元素重叠 >70%
- [ ] 首镜作为后续镜头的参考基准
- [ ] 图转视频仅描述运动，不改变视觉风格

---

## 💡 高级技巧

### 1. 批量生成时的一致性维护
- 使用同一个 seed 值（Stable Diffusion）
- 使用 --cref 参数链式引用（Midjourney）
- 第一个镜头用最高权重，后续降低变化层权重

### 2. 处理不一致的情况
- 如某个镜头偏差过大，提高固定层DNA权重至 1.6
- 使用 inpainting 局部修复不一致区域
- 重新生成时增加参考图的 --cw 权重值

### 3. 动态场景的处理
- 保持核心人物特征不变
- 场景DNA可以分为"环境DNA"和"临时场景"两部分
- 临时场景权重降低至 0.6-0.8
