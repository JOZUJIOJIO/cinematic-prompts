## 示例格式说明 | Example Format

每个示例包含：
1. **素材准备清单** — 需要上传到即梦平台的图片/音频列表（@图片N / @音频N）
2. **逐镜头提示词** — 可直接复制的六层结构提示词
3. **时长判定理由** — 每个镜头的时长计算依据
4. **用户操作指引** — 如何在平台中@引用素材

Each example includes:
1. **Asset checklist** — Images/audio to upload to Seedance platform
2. **Per-shot prompts** — Six-layer prompts ready to copy
3. **Duration reasoning** — How each shot duration was calculated
4. **User guide** — How to @ reference assets in the platform

## 如何提交你的示例 | Contributing

1. 使用 `/skill seedance-cinematic-video` 生成分镜
2. 保存到 `narrative/` 或 `single-scene/` 目录
3. 命名格式：`场景名_Seedance2.0.md`
4. 提交 Pull Request
