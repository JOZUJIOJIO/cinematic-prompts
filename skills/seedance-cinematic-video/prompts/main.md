# Seedance 2.0 电影级视频提示词引擎

从故事/剧本到可直接粘贴进即梦平台的电影级提示词。核心原则：**角色即变量、素材即槽位、时长由画面决定、提示词精确到每一帧的动作与声音。**

## 平台硬性约束

| 约束 | 值 | 备注 |
|------|-----|------|
| **单镜头时长** | **4-15秒** | 这是 Seedance 2.0 的硬上限，不可超过 |
| 图片参考上限 | 9张 | 角色/环境/道具/服装参考图 |
| 视频参考上限 | 3段，总时长≤15秒 | 运镜/动作参考 |
| 音频参考上限 | 3段，总时长≤15秒 | 对白/环境音/配乐 |
| 混合上传总上限 | 12个文件 | 图+视频+音频总计 |
| 视频比例 | 16:9 / 9:16 / 1:1 | — |
| 原生音画同步 | 口型同步、环境音效、BGM | 多语言支持 |
| 输出分辨率 | 1080p - 2K | — |

## 核心工作流

```
Step 1: 剧本分析 → 提取角色表、场景表、道具表
Step 2: 角色/素材注册 → 为每个角色和环境分配@槽位
Step 3: 智能分镜 → 拆镜头 + 根据画面复杂度判断时长(4-15s)
Step 4: 逐镜头生成提示词 → 角色用@占位、动作精确、音频精确
Step 5: 输出素材准备清单 + 带@占位的提示词
```

### Step 3: 智能分镜与时长判定

**拆镜头原则：**

1. 每个镜头 = 一个连续动作/事件
2. 场景切换 = 新镜头
3. 视角大幅变化 = 新镜头
4. 情绪转折点 = 新镜头

**时长智能判定规则（4-15秒）：**

| 画面类型 | 建议时长 | 判定依据 |
|---------|---------|---------|
| 环境空镜（无角色动作） | 4-6秒 | 无人物动作，纯氛围建立 |
| 单角色简单动作 | 5-8秒 | 一个角色完成一个动作 |
| 单角色复杂动作 | 8-12秒 | 一个角色完成连贯动作链 |
| 双角色互动 | 8-12秒 | 两个角色有交互动作 |
| 对白场景（需口型同步） | 按对白时长 | 对白朗读秒数 + 1-2秒缓冲 |
| 运镜幅度大（远→近/环绕） | 8-15秒 | 大范围运镜需要足够时间完成 |
| 快切蒙太奇 | 4-6秒 | 多个快速画面切换 |
| 情绪高潮/慢动作 | 10-15秒 | 需要时间渲染情绪 |

**判定公式：**

```
基础时长 = f(角色数, 动作复杂度, 运镜幅度)
+ 对白时长（如有口型同步）
+ 运镜附加（大幅运镜+3-5秒）

最终时长 = clamp(基础时长, 4, 15)
```

**输出格式：**

```markdown
| 镜头 | 内容摘要 | 判定时长 | 判定理由 |
|------|---------|---------|---------|
| 1 | 鹅城全景空镜 | 5秒 | 环境空镜，无角色动作 |
| 2 | 张麻子骑马入城 | 8秒 | 单角色+运镜跟拍 |
| 3 | 张麻子与黄四郎对话 | 12秒 | 双角色互动+对白 |
| 总计 | — | 25秒 | — |
```

### Step 4: 逐镜头提示词生成

**每个镜头的提示词必须包含以下六层：**

#### 层1: 角色层（用@占位符）

精确描述每个角色在本镜头中的**具体动作和表情**，用 `@图片N` 引用角色参考图。

```
@图片1 的角色（角色A）[具体动作]，[具体表情]，[具体姿态]
@图片2 的角色（角色B）[具体动作]，[具体表情]，[具体姿态]
```

**角色动作描述必须精确到：**
- 身体姿态（站/坐/走/跑 + 朝向）
- 手部动作（持物/指向/拍桌/举杯）
- 面部表情（微笑/愤怒/震惊 + 眼神方向）
- 角色间的空间关系（面对面/并排/一前一后）

#### 层2: 场景/环境层

```
在[具体环境]中，[环境细节]，[时间/天气]
```

如有场景参考图：`在 @图片N 的场景中`

#### 层3: 镜头语言层

```
[景别]，镜头[运镜方式]，[运镜速度和方向]
```

#### 层4: 光影/氛围层

```
[光线类型和方向]，[色调]，[氛围]
```

#### 层5: 音频层

Seedance 2.0 原生支持音画同步，提示词中必须明确音频需求：

```
环境音效: [具体环境音描述]
角色对白: @音频N 的对白，角色口型与语音精准对齐
话外音: [旁白内容描述]
配乐氛围: [音乐风格和情绪]
```

**音频描述规范：**

| 音频类型 | 提示词写法 |
|---------|----------|
| 环境音效 | `环境中有[风声/雨声/人群嘈杂/马蹄声]` |
| 角色对白（有音频） | `角色说出 @音频N 中的对白，口型与语音精准对齐` |
| 角色对白（无音频） | `角色说"[台词内容]"，口型自然` |
| 配乐 | `背景配乐为[风格]，@音频N 用于配乐` |
| 静默/留白 | `画面安静，仅有微弱的[环境音]` |

#### 层6: 约束层

```
画面流畅稳定不抖动，面部清晰不变形，细节丰富，
保持 @图片N 角色外观一致性，8K超高清，电影级质感
```

## 智能时长分析详细规则

### 动作复杂度评分

| 因素 | 低(+0s) | 中(+3s) | 高(+5s) |
|------|--------|--------|--------|
| 角色数 | 0-1人 | 2人 | — |
| 动作类型 | 静止/微动 | 行走/手势 | 打斗/奔跑/舞蹈 |
| 运镜幅度 | 固定/微推 | 横移/推拉 | 环绕/航拍/一镜到底 |
| 口型同步 | 无 | 短句(≤3秒) | 长对白(3-10秒) |
| 情绪强度 | 平静日常 | 紧张期待 | 爆发高潮 |

**计算方式：**

```
基础时长 = 4秒
+ 角色数加成
+ 动作加成
+ 运镜加成
+ 对白时长
= 原始时长

最终时长 = min(max(原始时长, 4), 15)
```

### 特殊场景时长参考

| 场景 | 推荐时长 | 说明 |
|------|---------|------|
| 开场环境建立镜头 | 5-8秒 | 航拍/远景+慢运镜 |
| 角色亮相/出场 | 6-10秒 | 需足够时间展示角色 |
| 对话（一句话） | 对白秒数+2秒 | 留出口型开合缓冲 |
| 对话（多句） | 拆分为多个镜头 | 每镜头≤15秒 |
| 动作戏/打斗 | 8-15秒 | 高动态需要时间展开 |
| 情绪渲染/慢镜头 | 10-15秒 | 慢动作拉长时间感 |
| 蒙太奇快切 | 4-8秒 | 多画面快速切换 |
| 结尾收束 | 8-12秒 | 后拉/升降，留白收束 |

## 素材槽位分配策略

**9张图片的推荐分配：**

| 角色/场景数 | 推荐分配 |
|-----------|---------|
| 2角色+1场景 | 角色×2 + 场景×1 + 道具/服装×2 + 首尾帧×4 |
| 3角色+2场景 | 角色×3 + 场景×2 + 道具×1 + 首尾帧×3 |
| 4角色+2场景 | 角色×4 + 场景×2 + 首尾帧×3 |
| 5角色以上 | 角色×5 + 场景×2 + 首尾帧×2（已接近上限） |

**超限处理：**
- 角色>5个：合并次要角色描述（不用参考图，用文字描述）
- 场景>3个：选最关键场景用参考图，其余用文字描述
- 优先保证主角的参考图槽位

## 质量检查清单

### 结构检查
- [ ] 角色表是否完整（每个角色有ID、描述、槽位）？
- [ ] 素材槽位总数是否≤12？图片≤9？视频≤3？音频≤3？
- [ ] 每个镜头时长是否在 **4-15秒** 之间？
- [ ] 时长判定是否有明确理由？

### 提示词检查
- [ ] 每个角色的动作是否精确到身体姿态+手部+表情+眼神？
- [ ] 角色间的空间关系是否明确？
- [ ] 是否只有一个核心动作（避免多动词混淆模型）？
- [ ] @占位符是否与素材槽位表一一对应？
- [ ] 音频层是否包含环境音效+对白/话外音+配乐？

### 输出检查
- [ ] 是否生成了素材准备清单（用户需要准备什么图/音频）？
- [ ] 提示词中的@占位符是否有用户操作指引？
- [ ] 镜头之间景别是否有变化？运镜节奏是否匹配叙事弧线？
