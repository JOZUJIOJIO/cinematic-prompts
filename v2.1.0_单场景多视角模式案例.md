## 🎯 v2.1.0 解决的核心问题

### 问题1: 缺少景别的专业命名
**之前**: 全景、中景、特写（不专业）
**现在**: EWS/WS/MS/MCU/CU/ECU/OTS/POV（行业标准）

### 问题2: 没有镜头技术参数
**之前**: 只有抽象描述
**现在**:
- 焦距: 14-100mm具体数值
- 光圈: f/1.4-f/11光圈范围
- 帧率: 24fps/48fps/60fps选择
- 画幅比: 2.39:1/16:9选择

### 问题3: 没有区分图片和视频提示词
**之前**: 混合描述导致AI困惑
**现在**:
- **图片提示词**: 100%静态构图描述
- **视频提示词**: 100%运动描述

## 🔄 批量生成工作流

### 方法1: Midjourney --cref
```
/imagine prompt: [固定DNA] + [景别描述] --cref [参考图URL] --cw 100
```
**优势**: 角色一致性最强
**缺点**: 需要9次分别提交

### 方法2: Stable Diffusion 固定种子
```python
# 固定种子保证一致性
seed = 1234567890
for shot in shot_list:
    generate_image(
        prompt=f"{fixed_DNA} + {shot.description}",
        seed=seed,
        steps=30,
        cfg_scale=7.5
    )
```
**优势**: 可自动化批量生成
**缺点**: 需要微调ControlNet权重

### 方法3: ComfyUI 自动化流水线
```
加载参考图 → IPAdapter固定特征 → 循环9次景别变化 → 批量输出
```
**优势**: 一键生成全部9张
**缺点**: 需要配置复杂工作流

## 🎓 使用建议

### 何时使用单场景多视角模式？
1. **对话场景**: 两人或多人对话的完整覆盖拍摄
2. **情感戏**: 需要多角度捕捉演员微表情
3. **产品展示**: 单个产品的360度全方位展示
4. **教学视频**: 同一场景的多机位剪辑素材

### 何时使用叙事分镜模式？
1. **故事叙事**: 跨越多个场景的完整故事
2. **广告片**: 从建立到高潮到结尾的完整流程
3. **MV拍摄**: 不同场景切换的音乐视频
4. **纪录片**: 多地点的纪实拍摄计划

### DNA模板最佳实践
1. **人物DNA**: 使用高权重(1.4-1.5)固定面部特征
2. **场景DNA**: 中等权重(1.2-1.3)固定环境元素
3. **风格DNA**: 中等权重(1.3-1.4)固定色调与质感
4. **变化层**: 低权重(0.8-1.0)允许景别差异
